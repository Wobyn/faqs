The gateway supports connections to these data sources on premises:

- BizTalk Server 2016
- DB2
- File System
- Informix
- MQ
- MySQL
- Oracle Database
- PostgreSQL
- SAP Application Server
- SAP Message Server
- SharePoint
- SQL Server
- Teradata
